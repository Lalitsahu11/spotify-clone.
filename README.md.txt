# Spotify Clone

A fully responsive Spotify clone built using **HTML**, **CSS**, and **JavaScript**.

## Features

- **Responsive Design**: Works well on both desktop and mobile devices.
- **Music Playback Interface**: Mimics Spotify's layout with album displays, playlist management, and navigation.
- **User Interface**: Clean and user-friendly, resembling the real Spotify interface.
- **Customizable Theme**: Future updates may include custom themes for the interface.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Flexbox / Grid Layouts
